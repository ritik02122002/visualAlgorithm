BUBBLE SORT

Bubble Sort is the simplest sorting algorithm that works by repeatedly swapping the adjacent elements if they are in the wrong order.

Problem Statement:

Given an array "arr" of integers. We have to sort the array "arr" in non-descreasing order using Bubble Sort(Assuming array indexing starts from 0)

Pseuodocode:

BubbleSort(Array arr, Size n)

Begin

for i = 0 to n-2 increment by 1 do // where, i is for the pass

for j = 0 to n – 2 – i increment by 1 do // this is for comparison

if arr[j] > arr[j+1] then

swap(arr[j],arr[j+1])

endif

endfor

endfor

End

Complexity Analysis:

Time Complexity:

Best Case: The best case occurs when the array is already sorted. So the number of comparisons required is n-1 and the number of swaps required = 0. Hence the best case complexity is O(n).

Average Case: It occurs when the array elements are in jumbled order that is not properly ascending and not properly descending. The average case time complexity of bubble sort is O(n*n).

Worst Case: The worst-case condition for bubble sort occurs when elements of the array are arranged in decreasing order. In the worst case, the total number of iterations or passes required to sort a given array is (n-1). where ‘n’ is the number of elements present in the array.Hence the worst case complexity is O(n*n).

Space Complexity:

O(1) as no additional space is used.
